# Vehicle Parking App

A simple vehicle parking management application that allows users to register, view available slots, and book parking.  

## 🚀 Features
- User signup / login
- View available parking slots
- Book and cancel bookings
- Admin page to view and manage slots

## 🛠 Tech stack
- Python, Flask
- SQLite
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Clone this repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
